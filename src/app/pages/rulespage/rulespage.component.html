Rules

  1. Start

    Press Start to have the game generate and display the first round.

  2. Sequence

    The game lights up one pad at a time (green, red, yellow, blue), adding one more step each round.

  3. Repetition

    You must replicate the sequence in the exact same order, pressing each pad once Simon has finished lighting it.

  4. Right vs. Wrong

    If you’re correct, you advance to the next round (and the sequence grows).
    If you’re wrong, you start over.

  5. Goal

    Reach as high a level as you can by memorizing the sequence; each completed level adds one point. Aim to hit the maximum level without failing!
